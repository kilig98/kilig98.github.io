MarkDown学习

1、标题

#一级标题

##二级标题

。。。

######六级标题

最多支持六级标题

2、文字

2.1删除线

使用“~~”：

~~删除线~~

实际效果：删除线

2.2斜体

使用“*”：

*斜体*

实际效果：斜体

2.3加粗

使用”**”

**加粗**

实际效果：加粗

2.4引用

使用“>”：

>"方方不是小方方"

实际效果：

“方方不是小方方”

可以使用多个“>”完成多级引用，也可以嵌套使用

2.5下划线

2.5.1 手动输入

<u>文字</u>

实际效果：文字

2.5.2 快捷键输入

使用“ctrl+u”输入

实际效果：文字

2.6高亮（需勾选扩展语法）

==文字==

实际效果：==文字==

2.7上下标（需勾选扩展语法）

上标：m^2^

实际效果：m^2^

下标：H~2~

实际效果：H2

3、表格

创建表格有两种方式：

3.1手动创建

使用“|”创建

| name | price|
| -----|------|
| beef |  8   |

实际效果如下：

name price
beef 8

3.2快捷键创建

使用“ctrl+t”快速创建表格

name price
chicken 8

4、列表

4.1无序列表

可以使用“*”、“+”、“-”创建无序列表

实际效果：

  • aaa
  • bbb
  • ccc

4.2有序列表

可以使用数字和’.’创建有序列表

实际效果：

  1. aaa
  2. bbb
  3. ccc

5、代码

5.1代码块

使用三个`创建代码块

public static void main(String[] args)&#123;

&#125;

5.2单行代码

使用一个`创建一条代码

java,c

6、分隔符

使用’***’，’—-‘，回车即可显示

实际效果：

7、跳转

7.1外部跳转

使用[提示文字](网页url)

实际效果：ACFUN

7.2内部跳转（文档内跳转）

使用[提示文字](#文档标题)

实际效果：跳转到标题

8、自动链接

使用<>

实际效果：<www.acfun.cn>

9、图片

使用![自己起的名字]（图片的位置）

或者直接复制到文档内就行
